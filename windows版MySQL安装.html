Windows版MySQL安装

2023-07-25 数据库 424 字
mysql

环境准备

服务器: windows 10

mysql版本: 5.7.23 X86, 64位

ZIP安装

下载压缩包

下载地址: https://dev.mysql.com/downloads/mysql/, 这里我们选择mysql5.7.23 X86,64位版本。

安装

  1. 解压缩到任意目录，这里我们解压到 D:\iTran\server\mysql-5.7.23-winx64

  1. 添加配置文件，由于在解压目录中并没有my.ini或my-default.ini，故在根目录添加my.ini并写入配置：

    [mysql]
    # 设置mysql客户端默认字符集
    default-character-set=utf8
    [mysqld]
    # 设置3306端口
    port=3306
    # 设置mysql的安装目录
    basedir=D:/iTran/server/mysql-5.7.23-winx64
    # 设置mysql数据库的数据的存放目录
    datadir=D:/iTran/server/mysql-5.7.23-winx64/data
    # 允许最大连接数
    max_connections=200
    # 允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统
    max_connect_errors=10
    # 服务端使用的字符集默认为UTF8
    character-set-server=utf8
    # 创建新表时将使用的默认存储引擎
    default-storage-engine=INNODB

  2. 初始化数据，在 bin 目录执行命令：

    mysqld --initialize --console

​ 控制台的 +/Xrz6;lh_h& 就是root的临时密码，后面再做改动

  1. 注册windows服务 

    mysqld --install MYSQL5.7

    卸载使用 mysqld --remove MYSQL5.7

  2. 启动

    net start MYSQL5.7

  3. 修改临时密码

    使用临时密码 +/Xrz6;lh_h& 登录

    mysql -u root -p

    修改密码为 Aa123456!

    ALTER USER USER() IDENTIFIED BY 'Aa123456!';

可能遇见的问题

  1. 初始化数据时提示缺少 MSVCR120.dll

​ 解决：下载驱动vcredist_x64.exe，下载地址: https://www.microsoft.com/zh-CN/download/details.aspx?id=40784

参考地址

  • 官方文档: https://dev.mysql.com/doc/refman/5.7/en/windows-install-archive.html
  • mysql-8.0.16-winx64.zip安装教程详解：https://zhuanlan.zhihu.com/p/48531203
